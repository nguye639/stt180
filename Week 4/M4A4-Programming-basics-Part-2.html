Pareto cumulative distribution function

The Pareto family of distributions is parameterized by \(\alpha > 1\) and \(x_0 \geq 1\) and has cumulative distribution function

\[ F(x) = \begin{cases} 0 & x < x_0\\ 1 - \left(\frac{x_0}{x} \right)^{\alpha - 1} & x \geq x_0. \end{cases} \]

For example with \(x_0 = 1\) and \(\alpha = 3\) the Pareto cumulative distribution function is

\[ F(x) = \begin{cases} 0 & x < 1\\ 1 - \left(\frac{1}{x} \right)^{2} & x \geq 1. \end{cases} \]

So for example if \(\alpha = 2.3\) and \(x_0 = 6000\) then the value of this function at \(x = 6700\) is

1 - (6000/6700)^(2.34 - 1)
[1] 0.1374537

and if \(\alpha = 3.5\) and \(x_0 = 1\) then the value of the function at \(x = 1.3\) is

1 - (1/1.3)^(3.5 - 1)
[1] 0.4810308

Write an R function called paretoCDF that takes as arguments a length one numerical vector alpha, a length one numerical vector x0, and a numerical vector x of length one or greater, and returns the cdf of the Pareto distribution with parameters alpha and x0 at x. paretoCDF = function(alpha,x0,x){
  
  pareto = 1 - (x0/x)**(alpha - 1)
  
  return(pareto)
}

Pareto quantile function

We can think of the cumulative distribution fuction answering the question, For a specific value of \(x\), what proportion of the data are less than or equal to \(x\). The quantile function is, roughly speaking, the inverse of the cumulative distribution function. It, again roughly speaking, answers the question, "For a specific value \(p\) between \(0\) and \(1\), what value of \(x\) has a proportion of exactly \(p\) of the data less than \(x\).

(The words roughly speaking are included because in some cases cumulative distribution functions are constant over certain intervals, and so dont actually have an inverse function. We wont concern ourselves with that possibility right now.)

  1. (This is a math question, not an R question.) Show that the quantile function \(Q\) for a Pareto distribution with parameters \(\alpha\) and \(x_0\) is given by
#1-f = (x0/x)^a-1
#(1-f)^1/a-1 = x0/x
#x0(1-f)^1/a-1 = 1/x
#x0(1-f)-1/a-1 = x, f = p,

\[ Q(p) = Q(p, \alpha, x_0) = {x_0}{(1-p)^{-\frac{1}{\alpha - 1}}} \quad\text{for}\quad 0 < p < 1. \]

  1. Write a function paretoQuantile with arguments alpha, x0, and p which returns the Pareto quantile function evaluated at p. Here p is a vector of length at least one, all of its values being between 0 and 1. For testing purposes, your function produce output that agrees with whats shown below.
paretoQuartile = function(alpha, x0, p){
  
  Q = x0*(1-p)**-(1/(alpha-1))
  return(Q)
}
